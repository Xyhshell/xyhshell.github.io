Mklink 软连接操作器

一款可以帮助你快捷搬运数据的软件，帮你腾出寸土寸金的储存空间😁

封装原理

运行环境

经过测试适用：windows 10 + / windows 7 （32/64bit）

功能 / 使用说明：(V2.1 版更新说明！)

V2.1版：

  • 优化部分逻辑和运行操作，界面显示
  • 优化部分变量的赋值
  • （本版新增）解除软连接（rmdir命令）：可解除已经设置软连接
  • 去除黑框

说明：

1.软件操作时都会在运行目录下将搬动的数据以zip格式的压缩包存档，
但不保证文件过大时（尤其时正在运行（使用）的文件数据）io冲突时产生zip的损坏，所以建议先备份在使用。

2.基于V2版修改，历史遗留问题【多线程】依旧未加入，所以搬运大文件时如果产生【程序无响应】提示，静心等待即可！！！

3.去除烦人的命令框，所以一些日志和报错就不会在显示（看不到就是没有bug）

历史日志：

非常简单

1、需要右键以管理员权限运行（mklink的原因)

（V2版，更新mklink /j 模式，不用管理员权限也可以操作！）

2、涉及压缩解压缩操作，没有弄线程，所有转换时依文件大小会有些许卡顿！

基本操作：

V2版，非管理员模式：mklink /j 管理员模式：mklink /d ）

1、获取源文件地址（需要折腾的文件夹）

2、获取新文件夹（需要保存的文件夹）

3、转移（原理：压缩源文件为zip, 删除源文件夹， 创建软链接， 解压缩至新文件夹）

至此完成所有操作！
