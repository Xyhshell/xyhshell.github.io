解决Windows蓝屏 代码SYSTEM_SERVICE_EXCPTION
(经验贴)

解决Windows蓝屏

蓝屏代码SYSTEM_SERVICE_EXCPTION

失败操作：rtf64x64.sys

解决办法

微软意见：打开

复现

每次开机进入系统时的锁屏界面均会导致蓝屏

由于无法进入系统，所以只能借助pe管理

仅本人测试

貌似时网卡驱动出问题导致的蓝屏

  1. 使用任意pe进入修复

  2. 使用 DISM++

  3. 在驱动管理中 将所有 带有 Realtek -> rtf64x64 层级关系的驱动卸载

  4. 千万不要禁用 带有 【rft】相关的服务项，会导致进入系统后可以联网但无法获取ip网络异常

以上结论来自【光影精灵8/3050Ti】

题后话

将网卡驱动卸载后，依旧可以联网，win11/10 会自加载驱动，可以不额外安装
